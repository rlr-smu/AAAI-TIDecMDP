# EventDecMDPs

## Initial Setup

```sh
git clone the repo
cd EventDecMDPs
mkdir Results
vim src/main/java/Config.java
```
Provide work directory path as a string:

```java
static String workDir = "/home/tarun/IdeaProjects/EventDecMdp/";
```

Create the following directories in your workDir:

```sh
mkdir Logs
mkdir experiements
mkdir Data
```
